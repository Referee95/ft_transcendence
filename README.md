## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/Referee95/ft_transcendence.git
cd ft_transcendence
```

### 2. Setup environment

```bash
cp .env.example .env
```

### 3. Run with Docker

```bash
docker-compose up --build
```

### 4. Access the app

```
http://localhost:3000
```
